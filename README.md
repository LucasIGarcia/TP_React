# Proyecto de Introducción a React con TypeScript

Este proyecto tiene como objetivo enseñar los conceptos básicos de React y TypeScript mediante una serie de módulos y ejemplos prácticos.

## Contenidos

1. **React**
2. **Introducción a React con TypeScript**
3. **Nuestro Primer Proyecto**
4. **Nuestro Primer Componente**
5. **Hook `useState`**
6. **Hook `useEffect`**
7. **Custom Hook `useForm`**
9. **App Final**

## Requisitos
- **Node.js** y **npm** instalados en el sistema.
- Editor de código como **VS Code** recomendado.
